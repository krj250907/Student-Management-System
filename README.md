# Student Management System (Java)

A simple console-based Student Management System developed using Java and Object-Oriented Programming (OOP).

## Features

- Add Student
- Display Student Details
- Menu-driven program
- Uses ArrayList for storage
- Object-Oriented Design

## Technologies Used

- Java
- Visual Studio Code
- Git
- GitHub
